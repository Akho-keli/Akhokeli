# Akho Keli Personal Brand Website

This is a static personal brand website for Akho Keli.

## Before publishing

Replace these placeholders in `index.html`:

- `https://YOUR-DOMAIN.com/`
- LinkedIn username
- Instagram username
- TikTok username
- GitHub username
- `YOUR-EMAIL@example.com`

Also replace the `YOUR PHOTO` placeholder with a professional photo if desired.

## Publish

You can upload these files to a static hosting provider such as GitHub Pages, Netlify, or Vercel.

## Google visibility

1. Publish the site on a real domain.
2. Verify the domain in Google Search Console.
3. Submit your sitemap.
4. Create a `sitemap.xml` using your real domain.
5. Keep your name and professional details consistent across LinkedIn and other public profiles.
6. Add genuine third-party mentions and press coverage over time.

Google decides independently whether enhanced search features or a Knowledge Panel appear.
